# 简易BBS论坛
  
## 环境
    SSM + shiro  
## bbs功能
    前台
        无账号权限
                    访问目录
                    注册及激活
        普通用户权限
                    访问bbs
                    主题
                        列表、搜索、发布、删除、回帖
        管理员权限
                    删主题、删回帖、禁言       
                                                      
    后台
        用户权限管理
        
    PS
        shiro 登陆权限控制
        MD5 邮件验证码限时一分钟激活注册
